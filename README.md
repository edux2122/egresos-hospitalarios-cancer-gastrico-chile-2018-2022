# egresos-hospitalarios-cancer-gastrico-chile-2018-2022
Preprocesamiento de microdatos DEIS-MINSAL para construcción de matriz de variables de egresos hospitalarios por cáncer gástrico (CIE-10: C16) en Chile, 2018–2022. Insumo para análisis de tendencias mediante regresión Joinpoint ajustada al Censo 2024.
